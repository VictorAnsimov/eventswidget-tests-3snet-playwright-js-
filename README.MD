Автоматизированные тесты для конструктора календаря 3Snet

Переключение языка (русский/английский)
Выбор тематики (7 вариантов + "Выбрать все")
Выбор стран (все страны + "Выбрать все")
Настройка размера (ширина 280-1020px, высота 240-720px)
Галочки "на всю ширину/высоту"
Цветовые темы (светлая/темная)
Генерация превью
Копирование HTML-кода
Кнопка "Очистить"
Скорость загрузки страницы
Реклама и пагинация

Установите зависимости
npm install
npx playwright install

Запустите тесты
npm test                      
npm run test:headed           
npm run test:debug
